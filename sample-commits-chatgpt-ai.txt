# Sample Commit Messages for ChatGPT AI

## Good Examples:
ChatGPT AI - Fix TypeScript compilation errors [AI]
ChatGPT AI - Optimize database query performance [AI]
ChatGPT AI - Add comprehensive error handling [AI]
ChatGPT AI - Improve code documentation [AI]
ChatGPT AI - Refactor component structure [AI]

## Bad Examples:
Fix bugs
Update code
Add features
(These won't trigger the webhook system)

## Remember:
- Always include [AI] in your commit messages
- Use descriptive commit messages
- Focus on one change per commit
- Test your changes before committing
